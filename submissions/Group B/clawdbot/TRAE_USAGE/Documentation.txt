# Prompt Trace Examples

## 1. Architecture Design Prompt

**Purpose**

Design the system architecture for an AI study application using Expo and AI services.

**Prompt**

```

Design a modular architecture for an Expo React Native application that:

* extracts text from PDFs
* extracts text from images using OCR
* allows editing extracted text
* generates quizzes using Google Gemini
* converts text into an audiobook reader
* uses Agora Conversational AI for high-quality TTS
* falls back to Expo Speech when offline

Provide a recommended folder structure and component architecture.

```

**Outcome**

Generated the base structure:

```

app/
hooks/
context/
server/

```

and helped define the data flow between extraction, editing, and study tools.

---

# 2. Quiz Generation Prompt

**Purpose**

Generate quizzes from extracted text using Gemini.

**Prompt Template**

```

You are an AI study assistant.

Generate quiz questions from the following text.

TEXT:
{extractedText}

Requirements:

* Include multiple choice questions
* Include true/false questions
* Include concept explanation questions
* Difficulty level: {difficulty}
* Number of questions: {depth}

Return JSON format:

{
"questions": [
{
"type": "multiple-choice",
"question": "",
"options": [],
"answer": ""
}
]
}

```

**Outcome**

Used inside:

```

hooks/useGemini.ts

```

for quiz generation.

# Prompt Engineering Notes

The following strategies improved AI output quality:

### Context Injection

Providing the chapter text improves tutor accuracy.

### Structured JSON Output

Quiz generation requires strict JSON format.

### Difficulty Controls

Quiz prompts include:

```

Difficulty: Easy | Medium | Hard
Depth: Number of questions

```
